Создай Telegram-бота на Python с использованием библиотеки python-telegram-bot.

Функционал бота:
- Бот должен предлагать полезные рецепты, которые вкусные и простые в приготовлении, не требуют много времени и сложных продуктов по запросу человека. Человек может запросить как составление рациона на день, так и на несколько дней. Бот должен учитывать предпочтения пользователя при формировании рецептов
Помимо составления рецептов бот должен уметь генерировать список покупок для данных рецептов
Также человек может написать те продукты, которые у него есть и бот должен придумать рецепты из данных продуктов.
- Должны быть доступны следующие команды: /рецептзавтрак - бот предлагает несколько простых рецептов для завтрака (сначала просто список, после выбора блюда дает рецепт), /рецептобед - бот предлагает несколько простых рецептов для обеда (сначала просто список, после выбора блюа дает рецепт), /рецептужин - бот предлагает несколько простых рецептов для ужина (сначала просто список, после выбора блюда дает рецепт), /рецептбыстрый - бот предлагает несколько простых и быстрых рецептов (сначала просто список, после выбора блюда дает рецепт), /менюдень - бот присылает несколько вариантов меню на день с разными рецептами (сначала просто список блюд, после выбора варианта меню присылает рецепты, /менюнеделя - бот присылает несколько вариантов меню на неделю, сначала просто список, /список (список продуктов из выбранных рецептов), /изчего (Рецепт из доступных продуктов, человек пишет какие продукты у него есть и бот присылает несколько вариантов блюд которые можно приготовить), /список покупок - для выбранного рецепта бот генерирует список покупок, /избранное - бот сохраняет рецепты в избранное, чтобы к ним можно было обратиться позже
- Как бот должен отвечать: Бот должен отвечать дружелюбно, коротко (без большого количества информации), структурированно
- [Какие данные хранить] - Бот хранит данные по избранным рецептам, данные по пользователю

Требования:
- Бот должен быть простым и понятным
- Код должен быть хорошо прокомментирован
- Использовать файл для хранения данных (JSON или SQLite)
- Добавить обработку ошибок
